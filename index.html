II Effects

Exercise 3 - Effects, Hide and Show:

[hide();] // [show();]

Zobacz co się stanie gdy klikniesz te przyciski:

jQuery( document ).ready(function($) {

var btnHide = $("#hide");

var btnShow = $("#show");

btnHide.on("click", function(e) {

e.preventDefault();

$("p").hide();

});

btnShow.on("click", function(e) {

e.preventDefault();

$("p").show();

});

});
